# BC Dining Diet Planner

A web app built to help Boston College students explore dining hall food options and align their meals with dietary goals and preferences.

This project was originally developed as a team project on Glitch and is mirrored here.

### Installation

1. Clone or download this repository
2. Navigate to the project directory:
   ```bash
   cd "BC Dining Diet Planner"
   ```

3. Start a local development server:
   ```bash
   python3 -m http.server 8000
   ```

4. Open your browser and navigate to:
   ```
   http://localhost:8000
   ```

## Features
- Browse food options by category
- View nutritional information from local JSON datasets
- Simple front-end built with HTML, CSS, and vanilla JavaScript

## Project Team

This was a collaborative project completed by a team of 4 students. Each team member contributed to different aspects including user research, design, development, and data collection.
